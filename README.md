# Анализатор тональности и частот текстов

Программа сама выгружает новости из RSS-лент (Лента.ру, РИА, Ведомости, РБК,
ТАСС), чистит их, считает самые частые слова и оценивает настроение каждой
новости, потом рисует графики. В качестве графики используется интерфейс командной строки.

## Какой язык за что отвечает

| Язык | Что делает |
|------|-----------|
| Python | выгрузка новостей из RSS, чистка и лемматизация, частоты, тональность, меню, облако слов, сохранение результатов |
| R | тесты Шапиро-Уилка и ANOVA, графики ggplot2 |

Python вызывает R через `rpy2` (R работает как библиотека внутри Python). Если
`rpy2` не установлен, программа сама запускает R через `Rscript`. Данные между
языками передаются автоматически через CSV в `output/tables`

## Установка

### Python, вариант A: PDM

1. Установка PDM
```bash
pip install pdm
```

2. Установка зависимостей

```bash
pdm install -G rpy2            # основной набор + rpy2 (связь с R)
```

Все Python-зависимости ставятся в папку `.venv` внутри проекта.

### Python, вариант B: pip

```bash
pip install -r requirements.txt
pip install rpy2              # основной способ связи с R
```

### R

1. PDM ставит только Python. Сам R это не pip-пакет, его нужно поставить отдельно.

2. Установка зависимостей (ggtplot2)
```bash
Rscript scripts/r/install_packages.R
# или прямо в R:  install.packages("ggplot2")
```

## Запуск

```bash
pdm run start        # через PDM
# или, если ставили через pip:
python start.py
```

После запуска появится меню:

```
  1. Загрузить данные
  2. Настроить параметры анализа
  3. Запустить анализ
  4. Инструкция (открыть PDF)
  5. Открыть папку с результатами
  0. Выход
```

### Пример запуска
`1` -> `1` (собрать новости со всех лент, нужен интернет) -> `3` (запустить).
Результаты будут находится в `output/`.

Пошаговая инструкция со скриншотами
есть в `notes/user_manual.pdf` (пункт меню `4`).

## Вывод

Графики в `output/plots/`: топ-15 слов (`top_words.png`), распределение
тональности (`sentiment_hist.png`), сравнение источников
(`sentiment_by_source.png`), динамика по дням (`dynamics.png`) и облако слов
(`wordcloud.png`).

Таблицы в `output/tables/`: тональность каждого текста (`documents.csv`),
частоты слов (`word_freq.csv`), сводка (`stats_summary.csv`) и результаты тестов
(`stat_tests.txt`).

## Ограничения

- Нужен интернет, без него новости не выгрузить.
- RSS отдаёт только свежие новости (за последние часы-дни), поэтому динамика получается за короткий период.
- Тональность считается по общему словарю (война, кризис, рост, победа и т.п.), а не нейросетью, так что на новостях это грубая оценка.
- Чистка и стоп-слова рассчитаны на русский язык.
- Для графиков нужен установленный R с пакетом `ggplot2`.
